1 Plots.jl

Plots(x,y) accepts 
1. two vectors (x,y) 
2. dataframe columns (df.x,df.y) or 
3. matrix columns M[:,1],M[:,2]

The values (x1,y1) represents the coordinates.

1.1 Accepting 2 vectors

# default plot is line plot
begin
  x = [1980,1990,2000,2010,2020]
    y1 = [40,45,47,51,53]
    y2 = [15,18,34,18,28]
    
    plot(x, [y1,y2],
         title="2 Line Plots",
         legend=false, 
         marker=:circle,
         size = (300,300))
end

1.2 Accepting 2 dataframe.columns

iris = dataset("datasets","iris")
names(iris)

scatter(iris.SepalLength, iris.SepalWidth,
       group=iris.Species,
       xlabel="SepalLength",
       ylabel="SepalWidth",
       size=(400,300))

1.3 Accepting 2 Matrix columns

# note: there's `xlimits, xlim and xticks`

M = [1 1; 1 2; 2 1; 2 2; 1.5 3]

scatter(M[:,1],M[:,2],
       framestyle=:origin,
       legend=false,
       xlabel="x",
         ylabel="y",
       xlimits=(0,4),
         ylimits=(0,4),
       size=(300,300))
