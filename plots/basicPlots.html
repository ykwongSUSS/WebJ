0.1 Basic Plots

Julia’s default Plots.jl is a good plot environment to use. The default backend is gr() and other backends are plotlyjs, pgfplots and pyplots. It also has recipes such as StatPlots and Distributions . This page lists basic plot examples.

  1. data entry
  2. basic plots (scatter, scatterlines, barplots, histograms, boxplot)

1.1 Scatter

using Plots
using TypedTables

trees |> Table
scatter(trees.girth, trees.height,
  title="Scatter Points",
  subtitle="Date: 20 May 2025",
  xlabel = "Girth",
  ylabel = "Height",
  group = trees.name,
  size = (300,300)
  )

1.3 BarPlot

using Statistics
# AvgHeight histogram of different trees

# split-apply-combine

treesGp = groupby(trees, :name) 
treesAvg = combine(treesGp, :height => mean)
treesAvg  |> Table
Table with 2 columns and 3 rows:
     name     height_mean
   ┌─────────────────────
 1 │ Elm      52.68
 2 │ Oak      53.05
 3 │ Sequoia  65.1333
# plot barPlot by trees.name

plot(bar(treesAvg.name, treesAvg.height_mean, 
         color=[:lightblue,:lightgreen,:orange], 
         legend=false, size=(300,300),
         title="Avg height of trees"))
